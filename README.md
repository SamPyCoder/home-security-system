# Home Security Monitoring System

A smart home security application designed to monitor visitors and detect known and unknown persons using camera surveillance and facial recognition. The system also enables users to report suspicious activities directly to law enforcement authorities through the application.
## Technologies Used
- Flutter
- Dart
- Python
- Django
- MySQL
- HTML, CSS, JavaScript
- JSON

## Modules
- User Module (Mobile App)
- Admin Module (Web)
- Police Module (Web)
- Camera Monitoring Module

## Features
- Visitor monitoring
- Known and unknown person detection
- Visitor log management
- Complaint and report system
- Security alerts for users
